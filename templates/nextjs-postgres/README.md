## Getting Started

First, install the dependencies:

```bash
pnpm install
# or
npm install
# or
yarn
```

Then, set up your environment variables by copying `.env.example` to `.env` and filling in the values.

Next, push your database schema:

```bash
npx prisma db push
```

Finally, run the development server:

```bash
pnpm dev
# or
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
